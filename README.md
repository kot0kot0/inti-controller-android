# Inti Controller (Android)

inti 光目覚まし時計を Android から自由に制御するための非公式クライアントアプリです。

## 特徴
- **公式アプリの消灯仕様回避**: 30分で消灯する公式アプリの仕様を回避して指定時間だけ点灯し続ける。

## ⚠️ 重要な警告
本アプリは公式アプリの制限を解除して動作します。
- **発熱への注意**: 長時間の点灯によりデバイスが熱を持つ可能性があります。異常を感じたら直ちに使用を中止してください。
- **自己責任**: 本アプリの使用による故障や事故について、開発者は一切責任を負いません。
- **非公式**: 本アプリは非公式であり、今後のデバイスのアップデート等により動作しなくなる可能性があります。
